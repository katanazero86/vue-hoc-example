# vue-hoc-example

- mixin : 컴포넌트에서 사용되는 중복되는 기능들을 하나로 묶어 재사용 가능하게 사용하는 방식
- HOC : React 에서 파생된 패턴이며, 컴포넌트에서 사용되는 기능을 재사용하기 위한 기술
(컴포넌트를 받아 컴포넌트를 반환한다)
